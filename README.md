# 🔒 ABAS — Sistema de Gestión Empresarial para Seguridad Privada

Sistema desktop desarrollado a medida para una empresa de seguridad privada en Honduras. Cubre todas las áreas operativas del negocio: recursos humanos, operaciones, planilla, inventarios y auditoría — en una sola plataforma integrada.

> *Estado:* ✅ En producción y uso diario por la empresa.

---

## 🛠️ Stack Tecnológico

| Tecnología | Uso |
|---|---|
| VB.NET (Visual Studio 2017) | Desarrollo de la aplicación desktop |
| SQL Server 2014 / 2019 | Base de datos relacional |
| ZeroTier One VPN | Acceso remoto seguro vía VPS privada |
| Report Viewer | Generación de reportes (+30 reportes) |

---

## 📦 Módulos del Sistema

### 👥 RRHH
- Registro de Oficiales y Aspirantes
- Gestión de documentos del personal

### 🗂️ Operaciones
- Asignaciones de personal
- Control de faltas y días libres

### 💰 Planilla & Pagos
- Generación automatizada de planilla
- Módulo de pagos integrado

### 🏪 Inventarios
Gestión completa con trazabilidad de movimientos:
- *Armería* — registro y movimiento de armas y munición
- *Uniformes* — asignación, devolución y movimientos
- *Equipo* — registro y movimientos
- *Mobiliario* — registro y movimientos
- *Otros* — proveedores, almacenes, vehículos

### 🏢 Clientes
- Registro de clientes y sucursales

### 📊 Dashboards
- Estadísticas y gráficas de estados operativos relevantes

### 🔍 Auditoría & Logs
- Log de inicios de sesión
- Log de notificaciones
- Log de incidencias operativas (faltas, días libres no respetados, etc.)

### 🔐 Usuarios & Acceso
- Asignación de módulos por usuario
- Niveles de acceso diferenciados
- Control operativo en almacén

---

## 📈 Alcance

- *+30 reportes* generados dentro del sistema
- Múltiples usuarios simultáneos con roles diferenciados
- Acceso remoto seguro sin exponer puertos públicos (ZeroTier One)
